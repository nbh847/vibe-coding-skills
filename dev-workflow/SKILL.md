# Dev Workflow

## 通用规则

- 使用中文与用户交互

## 后端项目

- 使用 venv 虚拟环境执行所有脚本
- 不安装额外依赖库，直接使用已安装的库

## 前端项目

- CSS 样式分类整理到合适的路径
- 自动识别项目结构，找到或创建合适的样式存放位置
- 仅整理当前正在编辑的页面的 CSS 样式
- 不修改已存在页面的样式（避免大规模重构）
